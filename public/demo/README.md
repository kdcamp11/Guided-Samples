# GRACE walkthrough demo

`grace-walkthrough.webm` is generated by the recorder and embedded in the
landing page's **"See how GRACE works"** section.

## Generate / refresh the video

```bash
npm install
npx playwright install chromium      # one-time
npm run record:walkthrough           # records against http://localhost:3000
```

Record against the deployed site instead:

```bash
GRACE_URL=https://graceofficial.app npm run record:walkthrough
```

The recorder drives the real product journey (Landing → New Project → Select
Your Route → Custom Apparel → Design Studio → Preview in Reality → Tech Pack →
Send to Production), paces each screen, hides dev overlays, and writes the
result to `public/demo/grace-walkthrough.webm`. It never alters the product flow.

Commit the regenerated `grace-walkthrough.webm` to publish it.
